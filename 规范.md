## 引入 vue 文件一定要加后缀名

## 组件选项的顺序

```js
export default {
  name: '',
  /*1. Vue扩展 */
  extends: '',
  mixins: [],
  components: {},

  /* 2. Vue数据 */
  props: {},
  model: { prop: '', event: '' },
  data() {
    return {}
  },
  computed: {},
  watch: {},

  /* 3. Vue资源 */
  filters: {},
  directives: {},

  /* 4. Vue生命周期 */
  created() {},
  mounted() {},
  destroy() {},

  /* 5. Vue方法 */
  methods: {}
}
```

## import 引入顺序

同等类型的放一起，建议优先放 `mixins` 和 `components`

- 第三方库
- mixins
- components
- api
- utils
